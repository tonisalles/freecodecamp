Quatro regras de design

Espaçamento

Uma regra simples que é quebrada o tempo todo. Você precisa organizar grupos de elementos semelhantes juntos e, em seguida, circundá-los com um espaço vazio. O espaçamento pode ser uma ferramenta surpreendentemente poderosa.

Grid

As dimensões dos elementos e as distâncias entre eles são organizadas em uma estrutura coesa conhecida como grade. Se um elemento ficar fora da grade, você saberá que algo está errado.

Fontes

Para essa regra, não deve haver mais do que três tipos diferentes de fontes em um único site. Uma para o cabeçalho, uma para o corpo do texto e uma terceira para destacar palavras essenciais.

Cores

Além da cor principal, também existem cores de destaque usadas para detalhes importantes em um site. Use as ferramentas certas para ajudá-lo a selecionar uma boa combinação de cores.
