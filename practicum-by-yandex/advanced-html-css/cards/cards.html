Espaçamento

Uma regra simples que é quebrada o tempo todo. Você precisa organizar grupos de elementos semelhantes juntos e, em seguida, circundá-los com um espaço vazio. O espaçamento pode ser uma ferramenta surpreendentemente poderosa.

Grid

As dimensões dos elementos e as distâncias entre eles são organizadas em uma estrutura coesa conhecida como grade. Se um elemento ficar fora da grade, você saberá que algo está errado.
